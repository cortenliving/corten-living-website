# Corten Living website starter

This is the first working website based on the approved **Option 2** mockup.

## Upload to GitHub

1. Download and unzip this folder.
2. Open your `corten-living-website` repository on GitHub.
3. Click **Add file → Upload files**.
4. Drag everything *inside* this folder into the upload area. Include the `assets` and `design-reference` folders.
5. Enter a message such as `Add first Corten Living website`.
6. Click **Commit changes**.

## Preview on your computer

Double-click `index.html`. The website will open in your browser.
